**symfonicat** is a JMS (JavaScript Management System) built on Symfony

It's taking a second to port from a built project into something that makes sense to be open-source and reusable. Just give me a minute and you're going to have the coolest thing you've ever seen, free. I built it, the name is tayp - don't wear it out :)

It's the web's perfect union of Symfony (Doctrine/PHP), Node.js/Webpack (JavaScript), and Bootstrap (SCSS). It is a heavily database driven system that handles routing, subdomains, Heroku/Cloudflare DNS automation, per-subdomain Electron applications, Webpack bundles, client-side routing, it's... well, it's awesome. It's better than anything that's out there - except for Drupal.

- if you don't like PHP, you can leave
- unless you like JavaScript more than PHP - you can stay
- Bootstrap people welcome, CSS3/SCSS/HTML5 are welcome here

better README.md coming soon but stay tuned for some RAD JavaScript API's that tie into Symfony in a beautiful swirl or functionality, reusability, and simplicity.
